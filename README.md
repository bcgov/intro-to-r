###intro-to-r

The repository contains lessons and resources for the one day "Introduction to R" designed to support skill development for the Environmental Stewardship Initiative. The workshop will be held in Smithers, British Columbia on March 2nd, 2020. 


Here, you will find: 
- Software installation instructions (R and R-studio) 
- Teaching material
- Demonstration code for broader R applications

---------

### Event Details 
- _What_: Introduction to r; what is it?  What does it do? How can it make your life easier? 
- _Where_: Office of Wet'suwet'en : 3873 1 Ave, Smithers, B.C.
- _When_: 8:30 am Monday March 2nd to 4:30 PM, 2020
- _Cost_: Free for participants


---------
